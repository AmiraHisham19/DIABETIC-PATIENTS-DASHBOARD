# DIABETIC-PATIENTS-DASHBOARD
## Overview
This interactive Excel dashboard presents a comprehensive analysis of diabetic patient data across 38,000+ cases. It integrates clinical metrics, treatment patterns, and hospital distribution to support data-driven decision-making in healthcare environments.

## Purpose
The dashboard was designed to:
- Simulate real-world clinical data for exploratory analysis.
- Identify trends in treatment prevalence, hospitalization, and readmission.

## Key Features
- **Demographics & Totals**: Breakdown of patient age, gender, and race categories.
- **Treatment Prevalence**: Pie chart showing usage rates of medications including insulin, metformin, sulfonylureas, and others.
- **Hospital Distribution**: Segmentation of inpatient, outpatient, and emergency encounters.
- **Length of Stay Analysis**: Bar chart comparing patient counts across hospitalization durations.
- **A1C Testing Rates**: Visual summary of diabetic monitoring compliance.
- **Readmission vs. Medication Load**: Comparative chart highlighting potential correlations.

## Technical Highlights
- Built using Excel Pivot Tables, Pivot Charts, and Slicers for dynamic filtering.
- Applied conditional formatting and custom visual alignment for clarity
